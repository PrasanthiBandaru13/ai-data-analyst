#  AI Data Analyst
🚀 Live Demo: https://ai-data-analyst-cjbmuddyvqf2bykqybpgxl.streamlit.app/
Ask questions in natural language and analyze your dataset.

## Features
- Upload CSV / Excel
- Automatic data analysis
- Built-in prompts (no AI needed)
- Optional AI (OpenAI / Ollama)

## Setup

### 1. Install dependencies
pip install -r requirements.txt

### 2. Run app
streamlit run app.py

## Example Queries
- Show top 10 rows
- Create histogram of 'age'
- Correlation heatmap

##  Tech Stack
- Python
- Pandas
- Streamlit
- LLM (optional)
